(1)基因依赖网络，见pone.0092023.pdf
这个方法在我们后面的文章中用实验验证过，该方法挺靠谱的，其思路是如果基因B与表型（预后）之间的关系，如果依赖基因A，则基因A与B就存在一条有向边。
使用的数据是基因表达数据和癌症病人预后数据。这个有现成的工具包，见：CMI_final.zip


(2)ceRNA网络，见：SYB2-8-96.pdf
其思路如下：因为miRNA的量有限，所以如果基因A与基因B受共同的miRNA调控的话，那么基因A与基因B的表达应该存在正相关关系。
原因：由于miRNA有限，A上调，那么该MiRNA调控基因A就多了，该miRNA就会变少，从而调控基因B的miRNA就变少了，而mirna是负调控基因的，则基因B也上调。
具体方法见论文，构建方法很简单，ceRNA网络2015年前后非常受关注，我们当时这个小论文应该是最早提出构建这种网络的方法之一。
使用的数据为：miRNA表达数据，基因表达数据，miRNA-gene关系数据（miRNA targets），有问题可随时讨论


(3)协同致死网络（A-B基因共同作用，影响癌症预后），发表在CCF B类会议上(理学院本科生为第一作者)，详细方法在lu2019.pdf,其代码可以在github找到：
https://github.com/lubo1220/Ensemble-Classifier-based-on-gene-synergistic-network.git

(4)DNA甲基化交互网络，本质上就是基因共表达网络，但是用在了DNA甲基化数据上，论文可见：s12859-019-2674-z.pdf. 这个可以直接用我们培训资料里面用的方法来做，即可以用到DNA甲基化数据，也可以用到基因表达数据。
代码直接使用我们的培训资料中的代码即可(co_net_construct_E.m).
